# AI Quant Research Hub - 详细开发计划

## 项目概述
基于 PRD 文档，建设一个面向投资顾问/量化分析师的 AI 驱动的股票量化研究平台，采用图形界面（Web Dashboard）进行可视化展示。

---

## 第一阶段：基础设施 (Week 1-2)

### 1.1 项目初始化
- [ ] 创建项目目录结构（参考现有 skills 架构）
- [ ] 完善 requirements.txt（添加 Flask/FastAPI、Web框架依赖）
- [ ] 创建 Docker 配置文件
- [ ] 配置日志系统

### 1.2 数据库设计
- [ ] 设计 MongoDB/PostgreSQL 数据模型
- [ ] 实现数据存储层 (storage.py 已有基础)
- [ ] 数据版本管理机制

### 1.3 后端 API 框架
- [ ] 选择 FastAPI 作为 Web 框架
- [ ] 设计 RESTful API 接口
- [ ] 实现用户认证（JWT）

## 第二阶段：核心模块开发 (Week 3-6)

### 2.1 数据采集模块 (skill_data)
- [ ] 股票历史行情抓取 (使用 baostock/tushare)
- [ ] 新闻/舆情数据采集
- [ ] 数据清洗与标准化
- [ ] 数据存储 (storage.py 扩展)

### 2.2 策略分析模块 (skill_ai)
- [ ] 因子模型实现（价值、成长、动量因子）
- [ ] 技术指标计算 (SMA, EMA, RSI, MACD)
- [ ] 财务指标分析
- [ ] 综合评分模型 (scorer.py 扩展)
- [ ] AI 策略分析 (analyzer.py 扩展)

### 2.3 风控模拟模块 (skill_risk)
- [ ] 回测引擎完善 (backtest.py)
- [ ] 风险指标计算：最大回撤、夏普比率、波动率
- [ ] 组合优化建议
- [ ] 风险警示生成

### 2.4 报告生成模块 (skill_report)
- [ ] PDF 报告生成 (使用 ReportLab/WeasyPrint)
- [ ] 图表可视化 (charts.py 扩展 - Plotly)
- [ ] 每日/周报模板
- [ ] 邮件/通知推送

### 2.5 系统运维模块 (skill_ops)
- [ ] 任务调度 (APScheduler)
- [ ] 心跳监控
- [ ] 日志管理
- [ ] 异常报警

## 第三阶段：图形界面开发 (Week 7-9)

### 3.1 UI 设计 (使用 Pencil)
**页面结构：**
1. **Dashboard 首页**
   - 市场概览（指数行情）
   - 今日推荐股票 Top 10
   - 风险指标仪表盘
   - 快捷操作入口

2. **股票分析页**
   - 个股行情走势图
   - AI 评分卡片
   - 因子分析雷达图
   - 新闻舆情时间线

3. **组合管理页**
   - 持仓组合列表
   - 组合收益曲线
   - 风险分布图

4. **回测分析页**
   - 回测参数设置
   - 收益对比图
   - 风险指标详情

5. **报告中心**
   - 报告列表
   - 报告预览/下载

### 3.2 前端实现
- [ ] 使用 Vue 3 + Vite 或 React
- [ ] 图表库：ECharts / Plotly.js
- [ ] UI 组件库：Element Plus / Ant Design
- [ ] 响应式布局
